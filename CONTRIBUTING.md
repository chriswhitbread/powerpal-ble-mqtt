# Contributing  

Thank you for considering contributing to this project! Please follow these guidelines:  

## Issues  
- Search existing issues before opening a new one.  
- Provide clear details and steps to reproduce.  

## Pull Requests  
- Fork the repository, create a new branch.  
- Follow the coding style used in the project.  
- Ensure secrets are not committed; use environment variables.  

## Documentation  
- Update README.md or docs files as needed.
